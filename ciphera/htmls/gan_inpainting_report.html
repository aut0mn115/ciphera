GAN 기반 결측 이미지 복원 및 분류 탐구 보고서

1. 연구 주제

GAN(Generative Adversarial Network)을 이용한 결측 이미지 복원(Inpainting) 모델 연구를 시작하려 했으나, 고등학생이 구현하기에는 난이도가 높아, 보다 간단한 형태로 변경하였습니다. 최종 주제는 "이미지 임베딩을 활용한 손상 이미지 분류 및 성능 분석"입니다.

2. 연구 목적

  • 이미지 임베딩(Image Embedding) 기법을 활용하여 손상 이미지와 원본 이미지를 분류하는 모델을 구현
  • 분류 성능을 시각적으로 분석하고, Confusion Matrix로 결과 비교
  • 테스트 데이터의 독립성을 유지하여 정확한 성능 평가 진행

3. 실험 데이터 구성

학습 데이터: 인형 사진 50장 촬영

  • original: 정상 이미지 40장
  • damaged: 손상 이미지 20장

테스트 데이터: 10장 촬영

  • original: 정상 이미지 6장
  • damaged: 손상 이미지 4장
dataset_v2/
 ├─ original/
 ├─ damaged/
 └─ test/
     ├─ original/
     └─ damaged/

4. 실험 방법

4.1 Orange 데이터마이닝 활용

  1. Image Import 위젯으로 학습 데이터와 테스트 데이터 불러오기
  2. Image Embedding 위젯에 SqueezeNet 모델 적용
  3. Logistic Regression 모델 학습
  4. Test & Score 위젯으로 모델 성능 평가 (Train/Test 분리 방식 적용)
  5. Confusion Matrix 위젯으로 결과 시각화

4.2 Train/Test 분리 방식

  • 테스트 데이터는 학습 과정에 절대 포함되지 않도록 분리
  • Train: 학습 데이터 Embedding
  • Test: 테스트 데이터 Embedding
[Image Import (학습 데이터)] → [Image Embedding] → [Logistic Regression] → [Test & Score] → [Confusion Matrix]
                                                          ↑
[Image Import (테스트 데이터)] → [Image Embedding]

5. 결과 및 분석

5.1 Confusion Matrix

Confusion Matrix를 통해 모델이 테스트 데이터의 Actual 값과 Predicted 값을 비교.

  • 모델의 분류 정확도를 확인 가능
  • 테스트 데이터의 라벨(Original/Damaged)과 예측값이 일치하는지 시각적으로 확인

5.2 성능 평가

  • Accuracy, Precision, Recall, F1-score 확인
  • 테스트 데이터가 독립적으로 유지되어 성능 평가의 신뢰성 확보

6. 결론 및 의의

이번 탐구를 통해:

  • 이미지 임베딩과 Logistic Regression을 결합한 손상 이미지 분류 모델을 구현함
  • Train/Test 분리 방식을 통해 테스트 데이터의 독립성을 확보
  • Confusion Matrix를 활용하여 분류 성능을 명확히 시각화함

의의

이 탐구는 GAN 기반 이미지 복원 연구로 확장 가능하며, 데이터 전처리, 임베딩, 분류 및 성능 시각화 전 과정을 이해하는 데 중요한 기초를 제공합니다.
