GAN 기반 결측 이미지 복원 및 분류 탐구 보고서

1. 연구 주제

GAN(Generative Adversarial Network)을 이용한 결측 이미지 복원(Inpainting) 모델 연구를 시도하려 했으나, 고등학생 수준에서의 구현 난이도를 고려하여 주제를 변경하였습니다. 최종 주제는 “이미지 임베딩을 활용한 손상 이미지 분류 및 성능 분석”입니다.

2. 연구 배경

딥러닝 기술의 발전으로 이미지 복원과 분류 분야는 급격히 성장하였습니다. 특히 이미지 임베딩을 활용하면 CNN 모델의 추상적 특성을 수치화하여 다양한 분류 문제에 응용할 수 있습니다. 본 연구는 Orange Data Mining을 통해 이미지 임베딩 기반의 손상 이미지 분류를 수행하였습니다.

3. 연구 목적

  • 이미지 임베딩(Image Embedding) 기법을 활용하여 손상 이미지와 원본 이미지를 분류하는 모델 구현
  • Confusion Matrix를 통해 모델 성능을 시각적으로 분석
  • Train/Test 분리 방식을 통해 테스트 데이터의 독립성 확보

4. 실험 준비

이번 실험에서는 인형 사진 총 60장을 직접 촬영하여 학습 데이터로 사용하였으며, 추가로 테스트용 이미지 18장을 별도로 촬영하였습니다. 그중 손상 이미지는 6장입니다.

5. 데이터 구성

dataset_v2/
 ├─ original/
 ├─ damaged/
 └─ test/
     ├─ original/
     └─ damaged/

학습 데이터와 테스트 데이터를 분리하여 모델 학습의 객관성을 확보하였습니다.

6. 실험 및 결과

Orange Data Mining에서 다음과 같은 워크플로우를 구성하였습니다.

[Image Import (학습 데이터)] → [Image Embedding] → [Logistic Regression] → [Test & Score] → [Confusion Matrix]
                                                           ↑
[Image Import (테스트 데이터)] → [Image Embedding]

Confusion Matrix를 통해 모델의 예측 결과를 시각적으로 확인하였으며, Accuracy, Precision, Recall, F1-score를 통해 성능을 평가했습니다.

7. 분석

모델은 손상 이미지와 원본 이미지를 비교적 높은 정확도로 분류하였습니다. Train/Test 분리 방식으로 인해 테스트 데이터의 독립성이 유지되어, 과적합(overfitting)의 위험을 줄였습니다.

8. 결론

본 탐구를 통해 이미지 임베딩을 기반으로 한 분류 과정을 이해하고, 데이터 전처리부터 성능 평가까지의 전체 흐름을 직접 경험하였습니다. 이 연구는 향후 GAN을 활용한 이미지 복원 연구로 확장할 수 있는 기초 자료로서의 의의를 가집니다.
